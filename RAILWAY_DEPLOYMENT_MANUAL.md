- GitHub pedirá para autorizar Railway App
- Clique em "Authorize"

- Clique "New Project"

```

POSTGRES_PORT=5432
POSTGRES_DB=neondb
POSTGRES_USER=neondb_owner
POSTGRES_PASSWORD=sua-senha
```

### 8️⃣ Aguarde Deploy
- Railway fará deploy automaticamente
- Leva ~5 minutos na primeira vez

### 9️⃣ Acesse Seu App
- Railway fornecerá uma URL única
- Ex: `https://mateu-coffee-production.up.railway.app/`
