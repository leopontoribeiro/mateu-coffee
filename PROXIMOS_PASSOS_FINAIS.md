### PASSO 2: Deploy via Railway

**No navegador:**

1. Vá para: https://railway.app/dashboard
2. Clique "+ New Project"
3. Clique "GitHub Repository"
4. Clique "Configure GitHub App" (ícone de engrenagem)
   - Isso abrirá GitHub
   - Clique "Authorize Railway App"
5. Volte para Railway
6. Procure por "mateu-coffee" na search
7. Clique no repositório
8. Railway faz deploy automático! 🎉
